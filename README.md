# pi_montecarlo

In this project I estimated π using Monte Carlo simulation.

# Installation

pip install -r requirements.txt

# Features

- Basic implementation
- Optimized vectorized NumPy implementation
- Visualization of random sampling
- Convergence analysis
- Performance benchmarking
- Runtime Comparison Plot

# Run

python basicversion.py
python optimisedversion.py
python visualisation.py
python convergenceanalysis.py
python benchmarking.py
python refinedbenchmarking.py
python runtimeplot.py
